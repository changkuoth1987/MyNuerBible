Luak Ba-bɛl in bäär
1Ɛn wa̱nɔ nɛy diaal wi̱i̱ muɔ̱ɔ̱n la ruackɛ kɛ thok kɛl. 2Kä mëë ci naath ɣöth kui̱c känyä ca̱ŋ cukɛ dhɔr mi car car jek ro̱o̱l Ci-nar, cukɛ cu cieŋ thi̱n. 3Kä cukɛ cu ruac kamnikiɛn i̱, “Bia, ta̱thnɛ rɔ tuɔpni kä banɛ kɛ cu bul.” Kä cukɛ duëlkiɛn täth kɛ tuɔpni kɛnɛ mun in la ruɛ̈thkɛ ni duël. 4Ɛ jɛn, cukɛ cu wee i̱, “Bia, kuanɛ rɔ ben lät kä wec kɛnɛ luak mi bäär mi bi wi̱cdɛ cop ni puɔ̱rä. Banɛ rɔ̱ moc kɛ kɛ̈k kɛ ɣöö /canɛ bi̱lɛ riaw rɛy ɣɔaa.” 5Ni ɛn wa̱nɔ cu Kuoth Nhial ben piny kɛ ɣöö bɛ wec kɛnɛ luak ëë ci naath ɛ täth ben gui̱l. 6Kä mëë cɛ jɛ ben nɛn, cuɛ cu wee i̱, “Nɛ̈nɛ jɛ, kɛn kɛ nɛy ti do̱r kɛl, kä kɛn diaal ruackɛ kɛ thok kɛl. Kä nɛmɛ ɛ jɛn tuk lätnikiɛn tin bi tuɔɔk, kä thilɛ mi bi kɛ lɛ dhal. 7Kuanɛ wä, banɛ thukiɛn wä nyon a /ca rɔ̱ bi̱lɛ liŋ kamnikiɛn.” 8Ɛn wi̱ni cu Kuoth Nhial kɛ cu riaw kä rɛy ɣɔaa kɛɛliw, kä cukɛ la̱t wecdiɛn päl. 9Kɛ kui̱ ɛmɔ cua wec ɛmɔ cɔl i̱ Ba-bɛl kɛ ɣöö ɛ jɛn gua̱a̱th ëë ci Kuoth Nhial ji̱ wec muɔ̱ɔ̱n diaal thukiɛn nyon thi̱n. Kä niɛ wi̱ni cu Kuoth Nhial kɛ jak kä bikɛ däk rɛy ɣɔaa kɛɛliw.
Gaat Cɛm
(1 Kuɛ̈n 1:24-27)
10Täämɛ, ti̱ti̱ kɛ gaat Cɛm. Kɛ kɔr kä mëë ci nyɔc thuuk kɛ kɔr runi da̱ŋ rɛw, guäth ëë ci Cɛm runkɛ a run ti kuɔ̱r kɛl, cuɛ Ar-pak-cad di̱eth. 11Kɛ kɔr däpä Ar-pak-cad cu Cɛm te te̱e̱kä kɛ run ti kur dhieec kɔ̱kiɛn, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
12Guäth ëë ci Ar-pak-cad tekɛ run ti jiɛn diɔ̱k wi̱cdɛ dhieec cuɛ Cɛla di̱eth. 13Kä kɛ kɔr däpä Cɛla cu Ar-pak-cad te te̱e̱kä kɛ run ti kur ŋuaan wi̱cdɛ diɔ̱k, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
14Mëë ci Cɛla runkɛ a run ti jiɛn diɔ̱k cuɛ E-bɛr di̱eth. 15Kä kɛ kɔr däpä E-bɛr cu Cɛla te te̱e̱kä kɛ run ti kur ŋuaan wi̱cdɛ diɔ̱k, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
16Mëë ci E-bɛr runkɛ a run ti jiɛn diɔ̱k wi̱cdɛ ŋuaan cuɛ Pɛ-lɛg di̱eth. 17Kä kɛ kɔr däpä Pɛ-lɛg cu E-bɛr lɛni te te̱e̱kä kɛ run ti kur ŋuaan wi̱cdɛ jiɛn diɔ̱k, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
18Mëë ci Pɛ-lɛg runkɛ a run ti jiɛn diɔ̱k cuɛ Riw di̱eth. 19Kä kɛ kɔr däpä Riw cu Pɛ-lɛg lɛni te te̱e̱kä kɛ run ti kur da̱ŋ rɛw wi̱cdɛ bäŋuan, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
20Mëë ci Riw runkɛ a run ti jiɛn diɔ̱k wi̱cdɛ rɛw cuɛ Thɛ-rug di̱eth. 21Kä kɛ kɔr däpä Thɛ-rug cu Riw lɛni te te̱e̱kä kɛ run ti kur da̱ŋ rɛw wi̱cdɛ bärɔw, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
22Mëë ci Thɛ-rug runkɛ a run ti jiɛn diɔ̱k cuɛ Na-ɣor di̱eth. 23Kä kɛ kɔr däpä Na-ɣor cu Thɛ-rug lɛni te te̱e̱kä kɛ run ti kur da̱ŋ rɛw ɛgi̱m, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
24Mëë ci Na-ɣor runkɛ a run ti jiɛn da̱ŋ rɛw wi̱cdɛ bäŋuan cuɛ Tɛra di̱eth. 25Kä kɛ kɔr däpä Tɛra cu Na-ɣor lɛni te te̱e̱kä kɛ run ti kuɔ̱r kɛl wi̱cdɛ jiath kɛl wi̱cdɛ bäŋuan, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth.
26Mëë ci Tɛra runkɛ a run ti jiɛn bärɔw cuɛ Ab-ram di̱eth kɛnɛ Na-ɣor kɛ Ä-ran.
Gaat Tɛra
27Täämɛ ti̱ti̱ kɛ kɛn gaat Tɛra. Tɛra cuɛ Ab-ram di̱eth kɛnɛ Na-ɣor kɛ Ä-ran. Kä cu Ä-ran Lɔt di̱eth. 28Cu kɔŋ li̱w ni Ä-ran kä gua̱n ni Tɛra dhɔɔr ëë diëthkɛ jɛ thi̱n ni Ur rööl Kɛ̈l-da-ya. 29Kä cu Ab-ram kɛnɛ Na-ɣor kuɛɛn. Cu ciek Ab-ram a Tha-ray, kä cu ciek Na-ɣor a Mil-ka nyiman Ith-ka tin la nyiee Ä-ran. 30Duundɛ ɣöö, cu Tha-ray a rol.
31Ɛ jɛn, cu Tɛra gatdɛ ni Ab-ram naŋ kɛnɛ gat döödɛ ni Lɔt gat Ä-ran, kɛ Tha-ray ciek gatdɛ ni Ab-ram, kä cuɛ jiɛɛn ni kɛɛl kɛ kɛ kä Ur ro̱o̱l Kɛ̈l-da-ya wä kɛ ro̱o̱l Kɛ-naan. Ku duundɛ la ɣöö mëë cikɛ cop ro̱o̱l Ɣa-ran cukɛ cieŋ ɛn wi̱ni. 32Cu ni̱n te̱e̱kä Tɛra a run ti kur da̱ŋ rɛw wi̱cdɛ dhieec, kä cuɛ cu li̱w ro̱o̱l Ɣa-ran.
