5
Gaat A-dam
1Mɛmɛ ɛ jɛn warɛgak ciöötni gan A-dam. Mëë cak Kuoth naath, cuɛ kɛ cak nikɛ cätdɛ. 2Cuɛ kɛ cak tuut kɛnɛ ŋuɔ̱ɔ̱t. Kä cuɛ kɛ poth kä cuɛ kɛ cɔl i̱ Naath. 3Kɛ gua̱a̱th ëë ci A-dam tekɛ run ti kuɔ̱r kɛl wi̱cdɛ jiɛn diɔ̱k, cuɛ gat di̱eth kä cuɛ jɛ cal kɛ cätdɛ pa̱ny, kä cuɛ ciötdɛ cɔl i̱ Thɛth. 4Kɛ kɔr däpä Thɛth cu run teekä A-dam a run ti kur bädäk ɛgi̱m. Kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 5Ɛ jɛ nɔ, täämɛ mat runi teekä A-dam diaal kɛ run ti kur bäŋuan wi̱cdɛ jiɛn diɔ̱k, cuɛ cu li̱w.
6Kä mëë ci Thɛth tekɛ run ti kuɔ̱r kɛl wi̱cdɛ dhieec, cuɛ I-nɔc di̱eth. 7Kɛ kɔr däpä I-nɔc cu run teekä Thɛth a run ti kur bädäk wi̱cdɛ bärɔw, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 8Ɛ jɛ nɔ, täämɛ mat runi teekä Thɛth diaal kɛ run ti kur bäŋuan wi̱cdɛ jiath kɛl wi̱cdɛ rɛw, cuɛ cu li̱w.
9Kɛ gua̱a̱th ëë ci I-nɔc tekɛ run ti jiɛn bäŋuan ɛ juc, cuɛ Kii-nan di̱eth. 10Kɛ kɔr däpä Kii-nan cu run te̱e̱kä I-nɔc a run ti kur bädäk wi̱cdɛ jiath kɛl wi̱cdɛ dhieec, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 11Ɛ jɛ nɔ, täämɛ mat runi te̱e̱kä I-nɔc diaal kɛ run ti kur bäŋuan wi̱cdɛ dhieec, cuɛ cu li̱w.
12Mëë ci Kii-nan lɛ tekɛ run ti jiɛn bärɔw ɛgi̱m cuɛ Maa-lä-lɛl di̱eth. 13Kɛ kɔr däpä Maa-lä-lɛl cu run te̱e̱kä Kii-nan a run ti kur bädäk wi̱cdɛ jiɛn ŋuaan, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 14Ɛ jɛ nɔ, täämɛ mat runi te̱e̱kä Kii-nan diaal kɛ run ti kur bäŋuan wi̱cdɛ jiath kɛl ɛgi̱m, cuɛ cu li̱w.
15Mëë ci Maa-lä-lɛl lɛ tekɛ run ti jiɛn bäkɛl wi̱cdɛ dhieec, cuɛ Yɛ-rɛd di̱eth. 16Kä kɛ kɔr däpä Yɛ-rɛd cu run te̱e̱kä Maa-lä-lɛl a run ti kur bädäk wi̱cdɛ jiɛn diɔ̱k, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 17Ɛ jɛ nɔ, cu mat runi te̱e̱kä Maa-lä-lɛl diaal a run ti kur bädäk wi̱cdɛ jiɛn bäŋuan wi̱cdɛ dhieec, kä cuɛ cu li̱w.
18Kä mëë ci Yɛ-rɛd tekɛ run ti kuɔ̱r kɛl wi̱cdɛ jiɛn bäkɛl wi̱cdɛ rɛw, cuɛ Ä-nɔk di̱eth. 19Kɛ kɔr däpä Ä-nɔk cu run te̱e̱kä Yɛ-rɛd a run ti kur bädäk ɛgi̱m, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 20Ɛ jɛ nɔ, cu mat runi te̱e̱kä Yɛ-rɛd diaal a run ti kur bäŋuan wi̱cdɛ jiɛn bäkɛl wi̱cdɛ rɛw, kä cuɛ cu li̱w.
21Kä kɛ gua̱a̱th ëë ci Ä-nɔk tekɛ run ti jiɛn da̱ŋ bäkɛl wi̱cdɛ dhieec, cuɛ Mɛ-thu-cɛ-la di̱eth. 22Kɛ kɔr däpä Mɛ-thu-cɛ-la cu Ä-nɔk ja̱l kɛɛl kɛnɛ Kuoth kɛ run ti kur diɔ̱k, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 23Ɛ jɛ nɔ, mat runi te̱e̱kä Ä-nɔk diaal kɛ run ti kur da̱ŋ diɔ̱k wi̱cdɛ jiɛn bäkɛl wi̱cdɛ dhieec. 24Cu Ä-nɔk ja̱l kɛɛl kɛnɛ Kuoth, kä cuɛ thil kɛ ɣöö ci Kuoth ɛ riw.
25Guäth ëë ci Mɛ-thu-cɛ-la lɛ tekɛ run ti kuɔ̱r kɛl wi̱cdɛ jiɛn bädäk wi̱cdɛ bärɔw cuɛ Lɛ-mɛk di̱eth. 26Kɛ kɔr däpä Lɛ-mɛk cu Mɛ-thu-cɛ-la te te̱e̱kä kɛ run ti kur bärɔw wi̱cdɛ jiɛn bädäk wi̱cdɛ rɛw, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 27Ɛ jɛ nɔ, cu mat runi te̱e̱kä Mɛ-thu-cɛ-la diaal a run ti kur bäŋuan wi̱cdɛ jiɛn bäkɛl wi̱cdɛ bäŋuan, cuɛ cu li̱w.
28Mëë ci Lɛ-mɛk tekɛ run ti kuɔ̱r kɛl wi̱cdɛ jiɛn bädäk wi̱cdɛ rɛw cuɛ gat mi dho̱o̱l di̱eth. 29Cuɛ ciötdɛ cɔl i̱ Nowa, laarɛ jɛ i̱, “Bi gat ɛmɛ kɔn lor lätnikɔn kɛnɛ cuɔ̱cdan muun in ci Kuoth Nhial ɛ bit.” 30Kɛ kɔr däpä Nowa cu Lɛ-mɛk te te̱e̱kä kɛ run ti kur dhieec wi̱cdɛ jiɛn bäŋuan wi̱cdɛ dhieec, kä cuɛ dhɔɔl kɛnɛ nyi̱er kɔ̱kiɛn lɛni di̱eth. 31Ɛ jɛ nɔ, mat runi te̱e̱kä Lɛ-mɛk kɛ run ti kur bärɔw wi̱cdɛ jiɛn bärɔw wi̱cdɛ bärɔw, cuɛ cu li̱w.
32Mëë ci Nowa tekɛ run ti kur da̱ŋ dhieec ɛgi̱m, cuɛ Cɛm kɛ Ɣɛ̈m kɛ Ya-pɛth di̱eth.
